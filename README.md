## 功能
* 管理音声
* 管理音声分类

## 环境

* node >= 18
* typescript >= 4

## 开发

包管理器请使用**pnpm**

### 安装&启动
```
pnpm install && pnpm dev
```

### 配置

#### 上传站配置
* 配置都在`src/configs`目录下

#### 项目配置
* 都在项目根目录下，包括`typescript`、`vite`、前端代理等相关配置

### 虚研社按钮后端
